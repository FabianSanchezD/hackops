# Backend (Express + Supabase)

Deploy notes (Render)
- Build Command: npm run build
- Start Command: npm start
- Environment: Node 22.x (matches local)
- Ensure the following env vars are set in the service:
  - SUPABASE_URL
  - SUPABASE_ANON_KEY
  - SUPABASE_SERVICE_ROLE_KEY
  - OPENAI_API_KEY
  
  Optional:
  - Google Meet's Keys
  - SendGrid API Key
